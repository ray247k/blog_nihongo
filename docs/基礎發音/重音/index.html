    日文當中，發音則是以聲音的高低來區別 以高低音的方式，代表不同單字的發音

    橋（はし）、筷子（はし）

    將音調分成

    • 高音：在假名上方畫記直線，代表該假名的發音為高音
    • 中音：在假名中間畫記直線，代表該假名的發音為中音
    • 低音：在假名下方畫記直線，代表該假名的發音為低音

    假名計算方式 #

    長音、促音、拗音也必須當作一個假名計算

    重音符號的表示 #

    在單字的後方會有一個框框，中間寫上 0~4 不等的數字 例如：【0】【1】【2】【3】

    隨著單字長度不同，也可能出現 5 號音、甚至 6 號音，不過這類單字數量較少

    【0】 #

    稱為 0 號音

    第一個假名為中音，第二個假名之後升為高音 若是單字只有一個假名，則直接發成高音

    は(樹葉)、わたし(我)、にほんご(日語)、さくらんぼ(櫻桃)

    【1】 #

    稱為 1 號音

    第一個假名為高音，第二個假名之後降為低音

    ほん(書)、てんき(天氣)、らいげつ(下個月)

    【2】 #

    稱為 2 號音

    第一個假名為中音，之後升高音，在第二個假名後降為低音

    たまご(雞蛋)、ひこうき(飛機)、マレーシア(馬來西亞)

    【3】 #

    稱為 3 號音

    第一個假名為中音，之後升高音，在第三個假名後降為低音

    せんせい(老師)、おてあらい(洗手間)、ぎんこういん(銀行員)

    【4】 #

    稱為 4 號音

    第一個假名為中音，之後升高音，在第四個假名後降為低音

    でんわばんごう(電話號碼)、おおさかじょう(大阪城)

    が #

    有些後方會標示「(が)」的單字，例如

    • おとうと(が)【4】弟弟
    • やすみ(が)【3】休假
    • はな(が)【2】花
    • き(が)【1】樹

    「(が)」代表單字後面接續的助詞，表示重音會在單字後面降為低音

    後方助詞經常用來辨別同音異義字，像是：は（歯、葉） 光看單字本身發音，有時無法判斷屬於哪個意思 這時只要看後方助詞的重音是否下降，就能判斷其屬於哪一種意思了。

    は（が）→ 無下降，屬於【0】，是樹葉的意思。 は（が）→ 有下降，屬於【1】，是牙齒的意思。

    規則 #

    1. 是幾號音就在第幾個假名之後將為低音
    2. 單字第一個假名，幾乎都是中音(除了 1 號音和一個假名的單字)
    3. 中音只會出現在字首
    4. 無論單字多長，在一個單字中，重音最多只會上升一次、下降一次
