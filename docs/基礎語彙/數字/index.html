    個位數念法 #

    • １ いち【2】
    • ２ に【1】
    • ３ さん【0】
    • ４ よん【1】し【0】
    • ５ ご 【1】
    • ６ ろく【2】
    • ７ なな【1】しち【2】
    • ８ はち【2】
    • ９ きゅう【1】く【1】
    • １０ じゅう【1】
    • ０ ゼロ【1】 れい【1】 まる【0】

    4、7、9的特殊唸法「し、しち、く」，一般用於時間及月份
    例：四月(しがつ)、七時(しちじ)、九時(くじ)、九月(くがつ)
    「０」有三種唸法，意思完全相同，只有使用時機之區別

    十位數唸法 #

    將個位數唸法累加起來，就成了十位數唸法

    13：10+3=じゅう＋さん＝じゅうさん 98：9+10+8=きゅう＋じゅう＋はち=きゅうじゅうはち

    更多 #

    億(おく)、兆(ちょう)的數字唸法同「万(まん)」

    100會唸作「百(ひゃく)」，不唸作「一百(いっぴゃく)」
    千也不唸作「一千(いっせん)」，唸作「千(せん)」

    但是在萬以上，就講成「一万(いちまん)、一億(いちおく)、一兆(いっちょう)」

    百位數以上的唸法，也是以累加的方式

    ３３７２１＝３００００＋３０００＋７００＋２０＋１ ＝さんまん＋さんぜん＋ななひゃく＋にじゅう＋いち

    三万三千七百二十一(さんまんさんぜんななひゃくにじゅういち

    小數點唸法 #

    小數點唸作「点（てん）」 。另外，0必須念成「れい」

    • ０．１： れい てん いち
    • ３４．５： さんじゅうよん てん ご

    年號 #

    西元 1998 年，中文唸成「１・９・９・８」，日文則唸 成「一千九百九十八」

    • 西元 1999 年：西暦 1999 年(せいれきせんきゅうひゃくきゅうじゅうきゅうねん)
    • 西元 2000 年：西暦 2000 年(せいれきにせんねん)

    建築物名 #

    若是建築物上有數字「０」時，一般習慣念作「まる(丸)」圓形的意思

    台北１０１ → 台北 いち まる いち １０９ビル → いち まる きゅう ビル

    錢幣 #

    日本錢幣分為紙鈔(札/さつ)和硬幣(玉/たま) 「一千、五千、一萬」為紙紗 「五百、一百、十圓、五圓、一圓」為硬幣

    「玉/たま」接在金額後面時，一般會唸成濁音「だま」

    • 一万円札(いちまんえんさつ)
    • 五千円札(ごせんえんさつ)
    • 千円札(せんえんさつ)
    • 五百円玉(ごひゃくえんだま)
    • 百円玉(ひゃくえんだま)
    • 五十円玉(ごじゅうえんだま)
    • 十円玉(じゅうえんだま)
    • 五円玉(ごえんだま)
    • 一円玉(いちえんだま)

    電話號碼 #

    電話號碼中，「０」可唸成「ゼロ」「まる」
    「0952-544333」的分隔線「-」則唸作「の」

    ２－２２３４５６８８ ゼロ に の に に さん よん ご ろく はち はち

    ０９５２－３３３－２２２ ゼロ きゅう ご に の さん さん さん の に に に
