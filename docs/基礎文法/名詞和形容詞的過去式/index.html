    之前已經學過名詞和形容詞的肯定以及否定用法
    稱為「現在肯定形」和「現在否定形」

    就和英文一樣，日文也有過去式！此篇筆記說明

    • 名詞和形容詞的過去式
    • 名詞和形容詞的過去否定式

    い形容詞 #

    い形容詞：以假名い結尾的形容詞

    否定句 #

    將字尾的「い」改成「く」後，加上「ありません」 #

    例句：

    \(ラーメンは\overset{おいしい}{美味}しい\textcolor{blue}{です}。\) (拉麵好吃)
    \(ラーメンは\overset{おいしい}{美味}しく\textcolor{blue}{ありません}。\) (拉麵不好吃)

    過去式 #

    將現在式語尾的「い」改為「かった」 #

    例句：

    \(\overset{いぬ}{犬}\textcolor{blue}{は}\overset{おお}{大}きいです。\) （狗很大隻）
    \(\overset{いぬ}{犬}\textcolor{blue}{は}\overset{おお}{大}きかったです。\) （狗以前很大隻）

    過去否定 #

    在「現在否定」句尾加上「でした」即可 #

    例句：

    \(\overset{いぬ}{犬}\textcolor{blue}{は}\overset{おお}{大}きく\textcolor{blue}{ありません}\) （狗不大隻）
    \(\overset{いぬ}{犬}\textcolor{blue}{は}\overset{おお}{大}きく\textcolor{blue}{ありませんでした}\) （狗以前不大隻）

    總結 #

    以「高い」為例

    現在 過去
    肯定 高いです。 高かったです。
    否定 高くありません。 高くありませんでした。

    な形容詞、名詞 #

    な形容詞：不以假名い結尾的形容詞

    「名詞」和「な形容詞」的肯定否定用法，完全一樣
    在文法上「な形容詞」和「名詞」的性質是類似的，因此文法上的變化也大致相同

    否定句 #

    直接加上「ではありません」 #

    例句：

    \(\overset{しず}{静}か\textcolor{blue}{です}。\) （安靜）
    \(\overset{しず}{静}か\textcolor{blue}{ではありません}。\) （不安靜）

    過去式 #

    將現在式句尾的です，改為でした即可 #

    例句：

    \(\overset{わたし}{私}\textcolor{blue}{は}\overset{がくせい}{学生}\textcolor{blue}{です}。\) （我是學生）
    \(\overset{わたし}{私}\textcolor{blue}{は}\overset{がくせい}{学生}\textcolor{blue}{でした}。\) （我以前是學生）

    過去否定 #

    在「現在否定」的句尾加上「でした」即可 #

    例句：

    \(\overset{わたし}{私}\textcolor{blue}{は}\overset{がくせい}{学生}\textcolor{blue}{ありません}。\) （我不是學生）
    \(\overset{わたし}{私}\textcolor{blue}{は}\overset{がくせい}{学生}\textcolor{blue}{ありませんでした}。\) （我以前不是學生）

    總結 #

    な形容詞 #

    以「有名めい」為例

    現在 過去
    肯定 有名です。 有名でした。
    否定 有名ではありません。 有名ではありませんでした。

    名詞 #

    以「学生」為例

    現在 過去
    肯定 学生です。 学生でした。
    否定 学生ではありません。 学生ではありませんでした。

    整理表格 #

    現在肯定 現在否定 過去肯定 過去否定
    名詞 学生です。 学生ではありません。 学生でした。 学生ではありませんでした。
    な形容詞 有名です。 有名ではありません。 有名でした。 有名ではありませんでした。
    い形容詞 高いです。 高くありません。 高かったです。 高くありませんでした。

    補充 #

    • ～ありません：正統文法，可用在口語會話、文章書信和正式場合中。
    • ～ないです：口語文法，一般常用在口語會話中。
    • ～じゃないです：女生一般不太使用結尾有「ない、ありません」等否定語氣的句子
