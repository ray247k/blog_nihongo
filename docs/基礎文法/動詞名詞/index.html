    動詞+名詞 #

    日文動詞，若是放在名詞前面、用來修飾名詞時，中間不用加上任何假名
    不需要中文「的」作為連接詞

    動詞 + 名詞 #

    名詞前方只能使用「動詞普通形（常体）」
    「動詞ます形」無法直接放在名詞之前！

    \(ご\overset{はん}{飯}を\overset{た}{食}べる\overset{ひと}{人}。\)
    （吃飯的人）

    \(\overset{ね}{寝}る\overset{ねこ}{猫}。\)
    （睡覺的貓）

    中文母語者經常會犯的錯誤：
    い形容詞接名詞時，中間也不能加「の」

    \(\overset{しろ}{白}いご\overset{はん}{飯}。\)
    \(おいしい\overset{りょうり}{料理}。\)
