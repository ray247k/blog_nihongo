    存在句型(在…有…) #

    在什麼地方有什麼東西

    助詞「に」可以用來表示「目的地」，也可以用來表示「東西存在的場所」

    • 有生命的東西：使用「います」
    • 無生命的東西：使用「あります」

    有生命、會動 #

    幽靈、妖怪也歸類在此類

    場所地點 + に + 東西 + がいます。 #

    例句 #

    \(\overset{かみばこ}{紙箱}\textcolor{blue}{に}\overset{ねこ}{猫}\textcolor{blue}{がいます}。\) （紙箱裡有貓）

    \(\overset{どうぶつえん}{動物園}\textcolor{blue}{に}ライオン\textcolor{blue}{がいます}。\) （動物園裡有獅子）

    \(\overset{きょうしつ}{教室}\textcolor{blue}{に}\overset{がくせい}{学生}と\overset{せんせい}{先生}\textcolor{blue}{がいます}。\) （教室裡有學生和老師）

    無生命、物品 #

    植物有生命但是不用動，歸在此類

    場所地點 + に + 東西 + があります。 #

    例句 #

    \(\overset{へや}{部屋}にベッド\textcolor{blue}{があります}。\) （房間裡有床）

    \(\overset{としょかん}{図書館}に\overset{ほん}{本}とパソコン\textcolor{blue}{があります}。\) （圖書館裡有書和電腦）

    \(\overset{ほんだな}{本棚}に\overset{しょうせつ}{小説}と\overset{ざっし}{雑誌}\textcolor{blue}{があります}。\) （書架上有小說和雜誌）

    動作地點(在…做…) #

    在某地方做某事

    場所地點 + で + 動詞 #

    例句 #

    \(部屋\textcolor{blue}{で}ご\overset{はん}{飯}を\textcolor{blue}{\overset{た}{食}べます}。\) （在房間吃飯）
    \(運動場\textcolor{blue}{で\overset{はし}{走}ります}。\) （在運動場跑步）
    \(ソファ\textcolor{blue}{で\overset{ね}{寝}ます。}\) （在沙發上睡覺）

    「に」vs「で」 #

    兩個都是「在」的意思

    後方是「あります、います」時，前面用「に」
    後方是其他動詞時，前面用「で」

    存在句型「～に～があります」 #

    用於表示「什麼東西在什麼地方」

    動作發生句型「～で～」 #

    用於表示「在什麼地方做什麼動作」

    存在句型、方位詞 #

    方位詞 #

    中文 日文
    上方 \(\overset{うえ}{上}【0】\)
    下方 \(\overset{した}{下}【0】\)
    前方 \(\overset{まえ}{前}【1】\)
    後方 \(ろ\overset{うし}{後}【0】\)
    ～之中 \(\overset{なか}{中}【1】\)
    旁邊 \(\overset{となり}{隣}【0】、そば【1】\)

    存在句型 #

    場所地點 + の + 方位詞 + に + 東西+があります。 #

    場所地點 + の + 方位詞 + に + 東西+がいます。 #

    例句 #

    \(\overset{にわ}{庭}\textcolor{blue}{の\overset{なか}{中}に}\overset{まつ}{松}があります。\) （庭院裡有松樹）
    \(\overset{つくえ}{机}\textcolor{blue}{の\overset{うえ}{上}に}\overset{ざっし}{雑誌}があります。\) （桌子上有雜誌）
    \(\overset{くるま}{車}\textcolor{blue}{の\overset{うし}{後}ろに}\overset{こねこ}{子猫}と\overset{こねこ}{子犬}がいます。\) （車子後面有小貓和小狗）
    \(\overset{せんせい}{先生}\textcolor{blue}{のとなりに}\overset{ちち}{父}と\overset{はは}{母}がいます。\) （老師的旁邊是父親和母親）
