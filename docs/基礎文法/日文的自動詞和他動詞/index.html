    什麼是自動詞和他動詞 #

    自動詞 #

    前面不需要加上表示動作對象的「受詞」
    等於英文的不及物動詞。意思就是動詞後面不用有東西

    只要「名詞＋動詞」，意思就可以通

    他動詞 #

    前面要加上表示動作對象的「受詞」
    英文的及物動詞。意思就是動詞後面要有東西

    在動詞後面要加上「受詞」，意思才完整

    自動詞：不需要加上其他詞彙
    他動詞：要加上其他詞彙，意思才完整

    統整與分類 #

    自動詞 #

    由「自己」進行該動作
    ＝「～が～になる」（自然變成～）

    他動詞 #

    藉由「他人力量（外力）」進行動作
    ＝「～を～にする」（將～變成～）

    自動詞 意思 他動詞 意思
    \(\overset{はい}{入}る\) 自己進去某場所 \(\overset{い}{入}れる\) 將…放進去
    \(\overset{で}{出}る\) 自己從某場所出去 \(\overset{だ}{出}す\) 將…拿出來
    \(\overset{こわ}{壊}れる\) 東西壞掉了 \(\overset{こわ}{壊}す\) 將東西弄壞
    \(\overset{き}{消}える\) 東西自己消失了 \(\overset{け}{消}す\) 將…消除掉
    \(\overset{たお}{倒}れる\) 某物自己倒下 \(\overset{たお}{倒}す\) 將某物弄倒

    自他動詞用法 #

    基本用法 #

    を ＋ 他動詞 #

    其他助詞 ＋ 自動詞 #

    例句：

    \(\overset{わたし}{私}は\overset{あさ}{朝}\textcolor{blue}{\overset{お}{起}きる}。\)
    （我早上起床）自動詞

    \(\overset{め}{目}が\textcolor{blue}{\overset{さ}{覚}める}。\)
    （眼睛睜開）自動詞

    \(\overset{ちょうしょく}{朝食}を\textcolor{blue}{\overset{た}{食}べる}。\)
    （吃早飯）他動詞

    \(\overset{くつ}{靴}を\textcolor{blue}{\overset{は}{履}く}。\)
    （穿鞋子）他動詞

    \(\overset{がっこう}{学校}に\textcolor{blue}{\overset{い}{行}く}。\)
    （去學校）自動詞

    \(バスに\textcolor{blue}{\overset{の}{乗}る}。\)
    （坐公車）自動詞

    \(\overset{ひと}{人}が\overset{おお}{多}くて\textcolor{blue}{\overset{こま}{困}る}。\)
    （人很多，很困擾）自動詞

    \(\overset{がっこう}{学校}に\textcolor{blue}{\overset{とうちゃく}{到着}する}。\)
    （到了學校）自動詞

    \(\overset{せんせい}{先生}に\textcolor{blue}{\overset{あ}{会}う}。\)
    （和老師見面）自動詞

    \(\overset{じゅぎょう}{授業}を\textcolor{blue}{\overset{う}{受}ける}。\)
    （上課）他動詞

    \(\overset{せんせい}{先生}の\overset{はなし}{話}を\textcolor{blue}{\overset{き}{聞}く}。\)
    （聆聽老師的話）他動詞

    \(\overset{しゅくだい}{宿題}を\textcolor{blue}{\overset{ていしゅつ}{提出}する}。\)
    （交作業）他動詞

    \(\overset{せんせい}{先生}が\overset{せんしゅう}{先週}のテストを\textcolor{blue}{\overset{くば}{配}る}。\)
    （老師發回上週的考卷）他動詞

    \(クラスメイトがすごく\textcolor{blue}{\overset{しんぱい}{心配}する}。\)
    （班上同學都非常擔心）自動詞

    例外狀況 #

    當表示「通過地點」時，自動詞前方也會用「を」 #

    例句：

    \(\overset{そら}{空}\textcolor{blue}{を}\overset{と}{飛}ぶ。\)
    （飛過天空。）

    \(\overset{みち}{道}\textcolor{blue}{を}\overset{わた}{渡}る。\)
    （穿越馬路。）

    \(\overset{ろうか}{廊下}\textcolor{blue}{を}\overset{はし}{走}る。\)
    （跑過走廊。）
