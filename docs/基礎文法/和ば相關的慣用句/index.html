    和「ば」相關的慣用句 #

    「～ば」的主要功能為表示假設
    必須先將該動詞變成「動詞ば形」後，才能使用

    愈…愈…：「～ば～ほど」 #

    由表示假設語氣的「ば」和表示程度的「ほど（程）」所組成

    後方經常接續「なる」這個動詞，表示變化的情況

    基本用法 #

    \(動詞ば形＋動詞原形＋ほど、～。\) #

    （愈…，愈…）

    例句：

    \(サッカーは\overset{れんしゅう}{練習}\textcolor{blue}{すれば\overset{れんしゅう}{練習}するほど}、\overset{じょうず}{上手}になる。\)
    （足球愈是練習，就愈是拿手）

    \(パソコンで\overset{しごと}{仕事}を\textcolor{blue}{すればするほど}、\overset{め}{目}が\overset{わる}{悪}くなる。\)
    （愈是使用電腦工作，視力就會愈差）

    \(\overset{うえ}{上}へ\textcolor{blue}{\overset{のぼ}{登}れば\overset{のぼ}{登}るほど}、\overset{きおん}{気温}が\overset{さ}{下}がる。\)
    （愈往上走，溫度愈低）

    \(\overset{やす}{休}みが\textcolor{blue}{\overset{なが}{長}ければ\overset{なが}{長}いほど}、クラスメートに\overset{あ}{会}いたくなる。\)
    （休假愈長，就會愈想念同學）
    => 接い形容詞

    \(\overset{さんかしゃ}{参加者}は\textcolor{blue}{\overset{おお}{多}ければ\overset{おお}{多}いほど}\overset{よ}{良}い。\)
    （參加者愈多愈好）
    => 接い形容詞

    \(\overset{へや}{部屋}が\textcolor{blue}{きれいであればきれいであるほど}、\overset{す}{住}み\overset{ごこ}{心地}がいい。\)
    （房間愈漂亮，住起來愈是舒服。）
    => 接な形容詞時，常會用「であれば」的形式

    …的話就好了、就可以了：「～ばいい」 #

    由表示假設語氣的「ば」和表示「好、不錯的」意思的「いい（良い）」所組成

    時常用於下達指示或是向對方詢問時

    基本用法 #

    \(動詞ば形＋いい（です）。\) #

    （…就好了）

    例句：

    \(６０\overset{てん}{点}を\overset{と}{取}れ\textcolor{blue}{ばいい}。\)
    （拿 60 分就好了）

    \(この\overset{みち}{道}を\overset{まっすぐ}{まっすぐ}\overset{い}{行}け\textcolor{blue}{ばいい}。\)
    （這條路直直走就可以了）

    \(コインを\overset{い}{入}れて、ここのボタンを\overset{お}{押}せ\textcolor{blue}{ばいい}。\)
    （放進硬幣後，按下按鈕就好了）

    \(\overset{いちにち}{一日}\overset{さんかい}{三回}、この\overset{くすり}{薬}を\overset{の}{飲}め\textcolor{blue}{ばいい}。\)
    （這個藥一天吃三次就好了）

    延伸用法 #

    反問 #

    \(動詞ば形＋いいんじゃない？\) #

    （…不就好了嗎？）

    質問 #

    語氣較為強烈

    \(動詞ば形＋いいよね！／いいでしょうね！\) #

    （…就可以了吧！）

    疑問 #

    \(動詞ば形＋いい？／いいですか？\) #

    （…就可以了嗎？）

    例句：

    \(\overset{べんきょう}{勉強}\textcolor{blue}{すればいい}。\)
    （只要讀書就好了）

    \(\overset{べんきょう}{勉強}\textcolor{blue}{すればいいんじゃない}？\)
    （只要讀書不就好了嗎？）=> 反問

    \(\overset{べんきょう}{勉強}\textcolor{blue}{すればいいよね}！\)
    （只要讀書就好了吧！）=> 質問

    \(\overset{べんきょう}{勉強}\textcolor{blue}{すればいい}？\)
    （只要讀書就可以了嗎？）=> 疑問

    只要…的話就能辦到：「～ばできる」 #

    由動詞ば形和「できる（能夠）」組成，常用來鼓勵他人

    基本用法 #

    \(動詞ば形＋できる。\) #

    （只要…的話就能辦到）

    例句：

    \(やれ\textcolor{blue}{ばできる}。\)
    （只要嘗試就能辦到）

    \(がんばれ\textcolor{blue}{ばできる}。\)
    （只要堅持就能辦到）

    \(\overset{ど}{努}\overset{りょく}{力}すれ\textcolor{blue}{ばできる}。\)
    （只要努力就能辦到）

    \(どうすれ\textcolor{blue}{ばできる}のか。\)
    （要怎麼做才能辦到呢？）

    \(\overset{じしん}{自信}を\overset{も}{持}っていれ\textcolor{blue}{ばできる}。\)
    （只要有自信就能辦到）

    \(\overset{ひま}{暇}があれ\textcolor{blue}{ばできる}。\)
    （只要有時間就能辦到）

    只要…的話就知道了：「～ばわかる」 #

    由動詞ば形和「わかる（知道）」組成，帶有些許輕視的感覺

    基本用法 #

    \(動詞ば形＋わかる。\) #

    （只要…的話就知道了）

    例句：

    \(\overset{み}{見}れ\textcolor{blue}{ばわかる}。\)
    （只要看就會知道）

    \(\overset{き}{聞}け\textcolor{blue}{ばわかる}。\)
    （只要問一下就會知道）

    \(\overset{い}{行}け\textcolor{blue}{ばわかる}。\)
    （只要去了就會知道）

    \(\overset{じっさい}{実際}に\overset{そう}{操}\overset{さ}{作}すれ\textcolor{blue}{ばわかる}。\)
    （實際操作就會知道）

    \(\overset{でんげん}{電源}を\overset{い}{入}れれ\textcolor{blue}{ば}\overset{こしょう}{故障}の\overset{もんだい}{問題}が\textcolor{blue}{わかる}。\)
    （只要按下電源開關，就會知道問題在哪裡）

    \(\overset{ほん}{本}を\overset{よ}{読}め\textcolor{blue}{ばわかる}。\)
    （只要讀書就會知道）

    …不就好了？：「～ば？」 #

    主要放在句尾，用以反問對方，特別常見於口語會話中
    其實是「～ばいいんじゃない？（…的話不就好了？）」的省略

    通常用於給予對方簡單的建議或提議時
    由於帶有較隨便的語氣，因此不可以對長輩或是在正式場合中使用

    基本用法 #

    \(句子＋動詞ば形？\) #

    （…不就好了？）

    例句：

    \(\overset{せんせい}{先生}に\overset{い}{言}え\textcolor{blue}{ば}？\)
    （跟老師說不就好了？）

    \(\overset{べんとう}{弁当}を\overset{か}{買}って\overset{く}{来}れ\textcolor{blue}{ば}？\)
    （去買便當來不就好了？）

    \(\overset{じぶん}{自分}で\overset{き}{決}めれ\textcolor{blue}{ば}？\)
    （自己決定不就好了？）

    \(\overset{トイレ}{トイレ}に\overset{い}{行}け\textcolor{blue}{ば}？\)
    （去上廁所不就好了？）

    \(\overset{ゆうき}{勇気}を\overset{だ}{出}して\overset{さそ}{誘}え\textcolor{blue}{ば}？\)
    （拿出勇氣去約他不就好了？）

    \(\overset{だいがく}{大学}に\overset{はい}{入}れ\textcolor{blue}{ば}？\)
    （進大學不就好了？）

    \(\overset{にほんご}{日本語}\overset{きょうし}{教師}になれ\textcolor{blue}{ば}？\)
    （當日文老師不就好了？）

    \(\overset{ちょ}{直}\overset{くせつ}{接}\overset{かれ}{彼}と\overset{はな}{話}せ\textcolor{blue}{ば}？\)
    （直接跟他說不就好了？）
