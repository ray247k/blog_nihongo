    こと的用法 #

    1. 轉換動詞成名詞，在名詞句中使用
    2. 對於某人表示喜好時

    一、使用「こと」將動詞名詞化 #

    名詞句定義：A（名詞）は B（名詞）です。 #

    如果 A 或是 B 不是名詞，可以使用「こと」來名詞化！
    具有名詞化功能的「こと」也會出現在其他常用句型和會話當中

    我的興趣是… #

    「興趣」和「看電影」都為名詞，因此屬於名詞句

    \(\overset{わたし}{私}の\overset{しゅ}{趣}\overset{み}{味}は（名詞）です。\) #

    \(\overset{わたし}{私}の\overset{しゅ}{趣}\overset{み}{味}は（動詞原形＋こと）です。\) #

    我的興趣是看電影（這件事情）

    兩種說法

    \(\overset{わたし}{私}の\overset{しゅ}{趣}\overset{み}{味}は\overset{えい}{映}\overset{が}{画}です。\) \(\overset{わたし}{私}の\overset{しゅ}{趣}\overset{み}{味}は\overset{えい}{映}\overset{が}{画}を\overset{み}{見}ることです。\)

    例句：

    \(\overset{わたし}{私}の\overset{しゅみ}{趣味}は\overset{りょうり}{料理}を作る\textcolor{blue}{こと}です。\)
    （我的興趣是做菜）

    \(\overset{わたし}{私}の\overset{しゅみ}{趣味}は\overset{ほん}{本}を読む\textcolor{blue}{こと}です。\)
    （我的興趣是看書）

    \(\overset{わたし}{私}の\overset{しゅみ}{趣味}は\overset{やきゅう}{野球}をする\textcolor{blue}{こと}です。\)
    （我的興趣是打棒球）

    我喜歡 ／ 討厭的事是… #

    \(\overset{わたし}{私}の\overset{す}{好}きは（名詞）です。\) #

    \(\overset{わたし}{私}の\overset{す}{好}きは（動詞原形＋こと）です。\) #

    我喜歡的（事情）是聽音樂（這件事情）

    \(\overset{わたし}{私}の\overset{す}{好}きなことは\overset{おんがく}{音楽}です。\)
    \(\overset{わたし}{私}の\overset{す}{好}きなことは\overset{おんがく}{音楽}を\overset{き}{聞}くことです。\)

    也可以將「 \(\overset{す}{好}き\) 」代替為其他形容詞

    \(\overset{わたし}{私}の\overset{きらい}{嫌い}な\textcolor{blue}{こと}はバスケットをする\textcolor{blue}{こと}です。\)
    （我討厭的事是打籃球）

    \(\overset{わたし}{私}の\overset{きらい}{嫌い}な\textcolor{blue}{\textcolor{blue}{こと}}は\overset{へや}{部屋}を\overset{そうじ}{掃除}する\textcolor{blue}{\textcolor{blue}{こと}}です。\)
    （我討厭的事是打掃房間）

    \(\overset{わたし}{私}の\overset{とくい}{得意}な\textcolor{blue}{こと}はギターを\overset{ひ}{弾}く\textcolor{blue}{こと}です。\)
    （我擅長的事是彈吉他）

    \(\overset{わたし}{私}の\overset{にがて}{苦手}な\textcolor{blue}{こと}は\overset{りょうり}{料理}を\overset{つく}{作}る\textcolor{blue}{こと}です。\)
    （我不擅長的事是做菜）

    二、對於某人表示喜好時 #

    對某人表示喜好時習慣在人名的後面加上「\textcolor{blue}{こと}」
    表示「關於對方全部的事」，常用於和某人告白時

    我們並不是只喜歡「那個人本身」，而是喜歡「所有和那個人相關的事」

    \(（人名）のことが\overset{す}{好}き / \overset{きら}{嫌}いです。\) #

    例句：

    \(\overset{わたし}{私}は彼女\textcolor{blue}{のことが}\overset{す}{好}きです。\)
    （我喜歡她）

    \(\overset{わたし}{私}はあの人\textcolor{blue}{のことが}\overset{きら}{嫌}いです。\)
    （我討厭那個人）

    \(\overset{わたし}{私}は\overset{ぶちょう}{部長}\textcolor{blue}{のことが}\overset{きら}{嫌}いです。\)
    （我討厭部長）

    \(私はおばあちゃん\textcolor{blue}{のことが}\overset{す}{好}きです。\)
    （我喜歡外婆）
