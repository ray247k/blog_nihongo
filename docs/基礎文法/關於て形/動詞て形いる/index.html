    動詞て形＋いる #

    動詞て形＋いる／います #

    「動詞て形+いる」主要有二種用法

    1. 現在進行式，相當於英文「V+ing」用法
    2. 表示從過去到現在一直持續的狀態

    現在進行式 #

    表示現在正在做什麼事

    \(\overset{わたし}{私}はご\overset{はん}{飯}を\overset{た}{食}べます\)
    \(\overset{わたし}{私}はご\overset{はん}{飯}を\textcolor{blue}{\overset{た}{食}べています}\)
    （我正在吃飯）

    \(\overset{わたし}{私}は\overset{しゅくだ}{宿題}をします\)
    \(\overset{わたし}{私}は\overset{しゅくだ}{宿題}を\textcolor{blue}{しています}\)
    （我正在寫作業）

    \(\overset{かれ}{彼}は\overset{およ}{泳}ぎます\)
    \(\overset{かれ}{彼}は\textcolor{blue}{\overset{およ}{泳}いでいます}\)
    （他正在游泳）

    \(\overset{はは}{母}は\overset{おこ}{怒}ります\)
    \(\overset{はは}{母}は\textcolor{blue}{\overset{おこ}{怒}っています}\)
    （媽媽正在生氣）

    \(\overset{いもうと}{妹}は\overset{ざっし}{雑誌}を\overset{よ}{読}みます\)
    \(\overset{いもうと}{妹}は\overset{ざっし}{雑誌}を\textcolor{blue}{\overset{よ}{読}んでいます}\)
    （妹妹正在看雜誌）

    表示持續狀態 #

    從過去一直持續到現在，沒有改變的狀態、習慣

    \(\overset{まいあさ}{毎朝}、\textcolor{blue}{\overset{さんぽ}{散歩}しています}。\)
    （我每天早上都習慣散步 => 從過去到現在都持續的狀態）

    \(\overset{まいにち}{毎日}バスで\textcolor{blue}{\overset{つうがく}{通学}しています}。\)
    （我每天都坐公車上學 => 從以前到現在都持續的狀態）

    \(\overset{わたし}{私}は\textcolor{blue}{\overset{けっこん}{結婚}しています}。\)
    （我已經結婚了 => 我從以前到現在都一直是結婚的狀態）

    \(あの\overset{かびん}{花瓶}は\textcolor{blue}{\overset{わ}{割}れています}。\)
    （那個花瓶從以前到現在一直都是破掉的狀態）

    ている和ました的區別 #

    日文 時態 注重的點
    ている 現在進行、持續 注重描述「一直持續」的動作、事情
    ました 過去式 注重描述「當下瞬間」發生的動作、事情

    例句：

    \(\overset{さん}{散}\overset{ぽ}{歩}しています。\)
    （我有散步的習慣）
    => 我從以前到現在，都一直有散步的習慣

    \(\overset{さん}{散}\overset{ぽ}{歩}ししました。\)
    （我之前有去散步）
    => 某個時間點有去散步，但是後來怎麼樣不知道，很有可能已經沒有去散步了

    \(\overset{けっこん}{結婚}しています。\)
    （我結婚了） => 我從以前到現在，都一直是結婚這個狀態

    \(\overset{けっこん}{結婚}しました。\)
    （我之前結婚了）
    => 之前某個時間點結婚了，但是後來怎麼樣不知道。一般講這句話時會認為是結婚後又離婚了

    \(そのことは\overset{し}{知}りました。\)
    （我之前知道那件事）
    => 不自然的用法：當下知道了那件事，但是後來怎麼樣不知道，很有可能已經忘記了
    \(そのことは\overset{し}{知}っています。\)
    （我知道那件事）
    => 我從以前到現在，都一直是知道這件事的狀態

    \(（\overset{かびん}{花瓶}が）\overset{わ}{割}れています。\)
    （花瓶破掉了） => 從之前到現在，這個花瓶一直都是破掉的狀態，還沒被修好

    \(（\overset{かびん}{花瓶}が）\overset{わ}{割}れました。\)
    （花瓶之前破掉了） => 當下花瓶破掉了，但是後來怎麼樣不知道。強調當下發生

    情境 #

    花瓶剛掉下去破掉：

    \(あ！\textcolor{blue}{\overset{わ}{割}れました}！\)
    （啊！破掉了！）

    花瓶掉下去破掉之後三天，朋友問起 #

    \(A: あの\overset{かびん}{花瓶}、どうなっていますか？\)
    （那個花瓶怎麼了呢）

    \(B: あの\overset{かびん}{花瓶}は...\textcolor{blue}{\overset{わ}{割}れています}...\)
    （那個花瓶…已經破掉了…）
