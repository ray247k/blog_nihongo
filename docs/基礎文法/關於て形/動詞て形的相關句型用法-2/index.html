    又遇到「動詞て形」了
    「動詞て形」是最重要的動詞變化之一，能衍生出相當多的相關句型

    此篇內容：

    1. 表示意料之外：動詞て形＋しまう
    2. 表示嘗試：動詞て形＋みる
    3. 表示事先：動詞て形＋おく

    啊！…了：「てしまう」 #

    表示「意料之外」或「遺憾」語氣

    通常說話者在使用時，聲音中會帶有吃驚的感覺

    通常對於已經發生的事情才會感到吃驚和遺憾
    因此「てしまう」常以過去式的形式「てしまった」進行使用

    基本用法 #

    「しまう」為第一類動詞，「ます形」為「しまいます」

    動詞て形＋しまう／しまった。 #

    （啊！…了）

    例句：

    \(\overset{わたし}{私}は\overset{こん}{今}\overset{げつ}{月}の\overset{おこ}{お小}\overset{づかい}{遣い}を\overset{ぜん}{全}\overset{ぶ}{部}\overset{つか}{使}って\textcolor{blue}{しまった}。\)
    （啊！我將這個月的零用錢全部用完了）

    \(\overset{ちち}{父}は\overset{いもうと}{妹}の\overset{つく}{作}ったケーキを\overset{た}{食}べて\textcolor{blue}{しまった}。\)
    （啊！爸爸吃掉妹妹做的蛋糕了）

    \(\overset{かのじょ}{彼女}はあの\overset{たか}{高}い\overset{ふく}{服}を\overset{か}{買}って\textcolor{blue}{しまった}のか。\)
    （啊！她買了那件很貴的衣服嗎？）

    \(\overset{ほ}{欲}しいオモチャは\overset{う}{売}り\overset{き}{切}れて\textcolor{blue}{しまった}。\)
    （啊！喜歡的玩具賣完了）

    \(\overset{かさ}{傘}を\overset{でん}{電}\overset{しゃ}{車}に\overset{わす}{忘}れて\textcolor{blue}{しまった}。\)
    （啊！把雨傘忘在電車裡了）

    口語習慣 #

    在口語會話中，為了方便發音，常會將「てしまう」唸成「ちゃう」的形式
    • 「てしまう」→「ちゃう」
    • 「でしまう」→「じゃう」

    例句：

    \(\overset{た}{食}べてしまう → \overset{た}{食}べ\textcolor{blue}{ちゃう}\)
    \(\overset{わす}{忘}れてしまう → \overset{わす}{忘}れ\textcolor{blue}{ちゃう}\)
    \(\overset{か}{買}ってしまう → \overset{か}{買}っ\textcolor{blue}{ちゃう}\)
    \(\overset{よ}{読}んでしまう → \overset{よ}{読}ん\textcolor{blue}{じゃう}\)
    \(\overset{よ}{呼}んでしまう → \overset{よ}{呼}ん\textcolor{blue}{じゃう}\)
    \(\overset{し}{死}んでしまう → \overset{し}{死}ん\textcolor{blue}{じゃう}\)

    …看看：「てみる」 #

    「みる」漢字寫作「見る（看）」，因此可以想成是中文的「…看看」。

    表示嘗試進行某動作

    基本用法 #

    「みる」為第二類動詞，「ます形」為「みます」

    動詞て形＋みる #

    （…看看）

    例句：

    \(あの \overset{あたら}{新}しいレストランに\textcolor{blue}{\overset{い}{行}ってみる}。\)
    （去那間新的餐廳看看）

    \(コンビニのインスタントコーヒーを\textcolor{blue}{\overset{の}{飲}んでみる}。\)
    （喝喝看便利商店的沖泡咖啡）

    \(\overset{せんせい}{先生}の\overset{て}{手}\overset{りょうり}{料理}を\textcolor{blue}{\overset{た}{食}べてみる}。\)
    （吃吃看老師親手做的料理）

    \(\overset{つうやく}{通訳}の\overset{しごと}{仕事}を\textcolor{blue}{してみる}。\)
    （做做看口譯的工作）

    其他用法 #

    除了「てみる」的形式外
    也可以使用過去式「てみた」、願望用法「てみたい」等等

    例句：

    \(あの \overset{あたら}{新}しいレストランに\overset{い}{行}って\textcolor{blue}{みる}。\)
    （去那間新開的餐廳看看）

    \(あの \overset{あたら}{新}しいレストランに\overset{い}{行}って\textcolor{blue}{みたい}。\)
    （想去那間新開的餐廳看看）表示願望

    \(あの \overset{あたら}{新}しいレストランに\overset{い}{行}って\textcolor{blue}{みた}が、おいしくなかった。\)
    （去了那間新開的餐廳，不過並不好吃）表示過去式

    \(\overset{つうやく}{通訳}の\overset{しごと}{仕事}をして\textcolor{blue}{みる}。\)
    （做做看口譯的工作）

    \(\overset{つうやく}{通訳}の\overset{しごと}{仕事}をして\textcolor{blue}{みたい}。\)
    （想做做看口譯的工作）表示願望

    \(\overset{つうやく}{通訳}の\overset{しごと}{仕事}をして\textcolor{blue}{みた}けど、とても\overset{たいへん}{大変}な\overset{しごと}{仕事}だった。\)
    （做了口譯的工作，真的是非常辛苦）表示過去式

    事先…：「ておく」 #

    「おく」的意思是「置く（放置）」

    表示預先做某樣動作，可以想成「將某件事先做起來放著」

    基本用法 #

    「おく」為第一類動詞，「ます形」為「おきます」

    動詞て形＋おく #

    （事先…）

    例句：

    \(\overset{よやく}{予約}\textcolor{blue}{しておく}。\)
    （事先預約旅館）

    \(\overset{れいぞうこ}{冷蔵庫}に\textcolor{blue}{\overset{い}{入}れておく}。\)
    （事先將啤酒放進冰箱中）

    \(\overset{あした}{明日}\overset{かいぎよう}{会議用}の\overset{しりょう}{資料}を\overset{じゅんび}{準備}\textcolor{blue}{しておく}。\)
    （事先準備好明天開會用的資料）

    \(\overset{やす}{休}みのことを\overset{せんせい}{先生}に\overset{い}{言}っ\textcolor{blue}{ておく}。\)
    （事先告訴老師要請假的事）

    \(\overset{かいしゃ}{会社}に\overset{い}{行}く\overset{まえ}{前}に、\overset{ひる}{昼}の\overset{べんとう}{弁当}を\overset{つく}{作}っ\textcolor{blue}{ておく}。\)
    （去上班之前，事先做好中午的便當）

    其他用法 #

    除了「ておく」的現在式外
    也可以使用過去式「てみた」、願望用法「てみたい」等等

    例句：

    \(ホテルを\overset{よやく}{予約}して\textcolor{blue}{おく／おきます}。\)
    （事先預約旅館）

    \(ホテルを\overset{よやく}{予約}して\textcolor{blue}{おいた／おきました}。\)
    （事先預約了旅館）表示過去式

    \(ホテルを\overset{よやく}{予約}して\textcolor{blue}{おきたい}。\)
    （想事先預約旅館）表示願望

    \(ホテルを\overset{よやく}{予約}して\textcolor{blue}{おこう／おきましょう}。\)
    （一起事先預約旅館吧）表示邀請

    \(ビールを\overset{れいぞうこ}{冷蔵庫}に\overset{い}{入}れて\textcolor{blue}{おく／おきます}。\)
    （事先將啤酒放進冰箱中）

    \(ビールを\overset{れいぞうこ}{冷蔵庫}に\overset{い}{入}れて\textcolor{blue}{おいた／おきました}。\)
    （事先將啤酒放進冰箱中了）表示過去式

    \(ビールを\overset{れいぞうこ}{冷蔵庫}に\overset{い}{入}れて\textcolor{blue}{おきたい}。\)
    （想先將啤酒放進冰箱中）表示願望

    \(ビールを\overset{れいぞうこ}{冷蔵庫}に\overset{い}{入}れて\textcolor{blue}{おこう／おきましょう}。\)
    （一起先將啤酒放進冰箱中吧）表示邀請

    口語習慣 #

    在口語會話中，為了發音方便
    會將「ておく」發音成「とく」、「でおく」則發音成「どく」的形式
    \(\overset{よやく}{予約}する → \overset{よやく}{予約}\textcolor{brown}{しておく} → \overset{よやく}{予約}\textcolor{blue}{しとく}\)
    \(\overset{い}{言}う → \overset{い}{言}っ\textcolor{brown}{ておく} → \overset{い}{言}っ\textcolor{blue}{とく}\)
    \(\overset{よ}{読}む → \overset{よ}{読}ん\textcolor{brown}{でおく} → \overset{よ}{読}ん\textcolor{blue}{どく}\)
