    • 了解助詞「も」「に」「で」「と」的基本用法

    も(也) #

    很常見到，用於表示兩者相等

    名詞 + も + 名詞／形容詞／動詞 + です #

    \(\overset{すずき}{鈴木}さんは\overset{だいがくせい}{大学生}です。私\textcolor{blue}{も}\overset{だいがくせい}{大学生}です。\)
    （鈴木先生是大學生，我也是大學生）

    \(ピザを\overset{か}{買}いました。パスタ\textcolor{blue}{も}かいました。\)
    （買了披薩，也買了義大利麵）

    に(表示目的地) #

    後面主要接續和「移動」有關的動詞，用來表示「移動的目的地」

    名詞は場所/地點+に+行(い)きます/来(き)ます/帰(かえ)ります #

    常見連接在：

    • \(\overset{い}{行}く\) ：去
    • \(\overset{く}{来}る\) ：來
    • \(\overset{かえ}{帰}る\) ：回家

    \(\overset{わたし}{私}は\overset{にほん}{日本}\textcolor{blue}{に}\overset{い}{行}きます。\) （我去日本）

    \(お\overset{きゃく}{客}さんは\overset{うち}{家}\textcolor{blue}{に}\overset{き}{来}ました。\) （客人來家裡了）

    \(\overset{おとうと}{弟}は\overset{いえ}{家}\textcolor{blue}{に}\overset{かえ}{帰}りました。\) （弟弟已經回家了）

    で(方法、手段、工具) #

    利用什麼方法、使用什麼東西做某事

    名詞 + は + 名詞 + で + 動作 #

    \(\overset{わたし}{私}はバス\textcolor{blue}{で}\overset{がっこう}{学校}に\overset{い}{行}きます。\)
    我搭(使用)公車去學校

    \(\overset{わたし}{私}はフォーク\textcolor{blue}{で}パスタを\overset{た}{食}べます。\)
    我用叉子吃義大利麵

    と(和) #

    跟名詞時用法一樣，表示兩人的連帶關係
    特別注意是接在名詞後面

    名詞 + は + 名詞 + と + 動作 #

    \(\overset{わたし}{私}は\overset{ちち}{父}\textcolor{blue}{と}\overset{しょく}{食}\overset{じ}{事}をしました。\)
    我和父親去吃飯

    \(\overset{わたし}{私}は\overset{おとうと}{弟}\textcolor{blue}{と}\overset{いもうと}{妹}\textcolor{blue}{と}\overset{じっか}{実家}に\overset{かえ}{帰}りました。\)
    我和弟弟妹妹回了老家

    句型比較 #

    用同樣的句子搭配不同的助詞來比較差異

    以「に」表示目的地

    \(\overset{わたし}{私}は\overset{にほん}{日本}\textcolor{blue}{に}\overset{い}{行}きます。\) （我去日本）

    以「も」表示「也」的意思

    \(\overset{わたし}{私}\textcolor{blue}{も}\overset{にほん}{日本}に\overset{い}{行}きます。\) （我也去日本）

    以「と」表示「和」的意思

    \(\overset{わたし}{私}は\overset{ともだち}{友達}\textcolor{blue}{と}\overset{にほん}{日本}に\overset{い}{行}きます。\) （我和朋友去日本）

    以「で」表示「交通工具」、「方法手段」

    \(\overset{わたし}{私}は\overset{ひこうき}{飛行機}\textcolor{blue}{で}\overset{にほん}{日本}に\overset{い}{行}きます。\) （我坐飛機去日本）

    以上四種是會話中經常用到的句型，務必完全理解其意思！
