    と(和) #

    用來連接名詞，屬於全部列舉

    名詞 + と 名詞 #

    \(\overset{わたし}{私}\textcolor{blue}{と}\overset{いもうと}{妹}\textcolor{blue}{と}\overset{おとうと}{弟}\) （我和妹妹和弟弟）

    \(\overset{いぬ}{犬}\textcolor{blue}{と}\overset{ねこ}{猫}\) （狗和貓）

    \(\overset{うめ}{梅}\textcolor{blue}{と}\overset{さくら}{桜}\) （梅花和櫻花）

    や…など(像是…等等) #

    表示從眾多物品中選擇幾樣當作例子，屬於部分列舉

    名詞 + や + 名詞 + など #

    \(\overset{わ}{和}\overset{しょく}{食}と\overset{い}{言}えば、\overset{すし}{寿司}\textcolor{blue}{や}\overset{みそしる}{味噌汁}\textcolor{blue}{や}てんぷら\textcolor{blue}{など}です。\)
    （說到和食，有壽司、味噌湯、天婦羅等等。）

    \(\overset{の}{飲}み\overset{もの}{物}と\overset{い}{言}えば、コーラ\textcolor{blue}{や}ジュース\textcolor{blue}{や}お\overset{ちゃ}{茶}\textcolor{blue}{など}です。\)
    （說到飲料，有可樂、果汁、茶等等）

    か(或者) #

    表示從名詞中擇一，屬於全部列舉

    名詞 + か + 名詞 #

    \(いつが\overset{ひま}{暇}ですか。\) （什麼時候有空呢？）
    \(\overset{どようび}{土曜日}\textcolor{blue}{か}\overset{にちようび}{日曜日}です。\) （星期六或是星期日。）

    \(\overset{なに}{何}を\overset{た}{食}べますか。\) （要吃什麼呢？）
    \(\overset{ていしょく}{定食}\textcolor{blue}{か}うどん\textcolor{blue}{か}ラーメンです。\) （定食或烏龍麵或拉麵。）

    綜合用法 #

    \(\overset{なに}{何}を\overset{た}{食}べますか。\) （要吃什麼呢？）

    沒有明確指定吃哪一樣 #

    像是定食、烏龍麵、拉麵等等吧。也可以是選項外的選擇

    \(\overset{ていしょく}{定食}\textcolor{blue}{や}うどん\textcolor{blue}{や}ラーメン\textcolor{blue}{など}です。\)

    明確表示從中挑選一樣 #

    定食或烏龍麵或拉麵中選一個

    \(\overset{ていしょく}{定食}\textcolor{blue}{か}うどん\textcolor{blue}{か}ラーメンです。\)

    我全都要 #

    定食和烏龍麵和拉麵全都要吃

    \(\overset{ていしょく}{定食}\textcolor{blue}{と}うどん\textcolor{blue}{と}ラーメンです。\)
