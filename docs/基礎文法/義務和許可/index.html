    義務和許可 #

    表示義務的句型「～なければなりません」
    表示許可的句型「～なくてもいいです」
    表示許可的句型 「～てもかまいません」「～なくてもかまいません」

    必須做… 不得不做… #

    表示責任、義務

    「動詞ない形」去掉「い」＋「ければなりません」 #

    \(\overset{がっこう}{学校}に\overset{い}{行}く → \overset{がっこう}{学校}に\textcolor{blue}{\overset{い}{行}かない}\)

    \(\overset{がっこう}{学校}に\textcolor{blue}{\overset{い}{行}かなければなりません}\)
    （不去學校不行）

    \(\overset{しごと}{仕事}をする → \overset{しごと}{仕事}を\textcolor{blue}{しない}\)

    \(\overset{しごと}{仕事}を\textcolor{blue}{しなければなりません}\)
    （不工作不行）

    \(\overset{うち}{家}に\overset{かえ}{帰}る → \overset{うち}{家}に\textcolor{blue}{\overset{かえ}{帰}らない}\)

    \(\overset{うち}{家}に\textcolor{blue}{\overset{かえ}{帰}らなければなりません}\)
    （不回家不行）

    \(12\overset{じ}{時}に\overset{ね}{寝}る → 12\overset{じ}{時}に\textcolor{blue}{\overset{ね}{寝}ない}\)

    \(12\overset{じ}{時}に\textcolor{blue}{\overset{ね}{寝}なければなりません}\)
    （不十二點時睡覺不行）

    和表示原因的「から」一起使用 #

    \(\textcolor{blue}{\overset{しごと}{仕事}をする}\)

    \(\overset{かいしゃ}{会社}に\overset{はい}{入}ったから、\textcolor{blue}{\overset{しごと}{仕事}をしなければなりません}。\)
    （因為已經進了公司，因此不工作不行）

    \(\textcolor{blue}{\overset{うち}{家}に\overset{かえ}{帰}る}\)

    \(もう 11\overset{じ}{時}\textcolor{red}{だから}、\textcolor{blue}{\overset{うち}{家}に \overset{かえ}{帰}らなければなりません}\)
    （因為已經十一點了，因此不回家不行）

    \(12 \textcolor{blue}{\overset{じ}{時}に\overset{ね}{寝}る}\)

    \(\overset{あした}{明日}は\overset{そつ}{卒}\overset{ぎょう}{業}\overset{りょこう}{旅行}\textcolor{red}{だから}、\textcolor{blue}{12\overset{じ}{時}に\overset{ね}{寝}なければなりません}\)
    （因為明天是畢業旅行，因此十二點不睡覺不行）

    不… 也可以 #

    表示許可、允許
    是先前學過的「～てもいいです」句型的變化版本

    「動詞ない形」去掉「い」＋「くてもいいです」 #

    原形 ない形 許可句 解釋
    \(\overset{よ}{読}む\) \(\overset{よ}{読}まない\) \(\overset{よ}{読}\textcolor{blue}{なくてもいい}\) 不讀也可以
    \(\overset{い}{言}う\) \(\overset{い}{言}わない\) \(\overset{い}{言}\textcolor{blue}{なくてもいい}\) 不說也可以
    \(\overset{た}{食}べる\) \(\overset{た}{食}ない\) \(\overset{た}{食}\textcolor{blue}{なくてもいい}\) 不吃也可以
    \(\overset{おぼ}{覚}える\) \(\overset{おぼ}{覚}えない\) \(\overset{おぼ}{覚}え\textcolor{blue}{なくてもいい}\) 不記也可以
    \(する\) \(\textcolor{blue}{しない}\) \(\textcolor{blue}{しなくてもいい}\) 不做也可以
    \(\overset{く}{来}る \) \(\textcolor{blue}{\overset{こ}{来}ない}\) \(\textcolor{blue}{\overset{こ}{来}ない}\) 不來也可以

    即使…也沒關係 #

    和「～てもいいです」很類似，但是感覺起來較為委婉有禮貌

    「動詞て形」＋「も」＋「かまいません」 #

    句型的差別，在後面的部分

    • いいです：表示「可以」
    • かまいません：表示「沒關係」(かまわない)
    原形 て形 許可句 解釋
    \(\overset{よ}{読}む\) \(\overset{よ}{読}\textcolor{blue}{んで}\) \(\overset{よ}{読}\textcolor{blue}{んでもかまいません}\) 讀了也沒關係
    \(\overset{い}{言}う\) \(\overset{い}{言}\textcolor{blue}{って}\) \(\overset{い}{言}\textcolor{blue}{ってもかまいません}\) 說了也沒關係
    \(\overset{た}{食}べる\) \(\overset{た}{食}\textcolor{blue}{て}\) \(\overset{た}{食}べ\textcolor{blue}{てもかまいません}\) 吃了也沒關係
    \(\overset{おぼ}{覚}える\) \(\overset{おぼ}{覚}え\textcolor{blue}{て}\) \(\overset{おぼ}{覚}え\textcolor{blue}{てもかまいません}\) 記住也沒關係
    \(する\) \(\textcolor{blue}{して}\) \(\textcolor{blue}{してもかまいません}\) 做了也沒關係
    \(\overset{く}{来}る\) \(\textcolor{blue}{\overset{き}{来}て}\) \(\textcolor{blue}{\overset{き}{来}てもかまいません}\) 來也沒關係

    即使不做… 也沒關係 #

    「動詞ない形」＋「～てもかまいません」＝「～なくてもかまいません」(多了否定語氣)

    • ～てもかまいません：即使做… 也沒關係
    • ～なくてもかまいません：即使不做… 也沒關係

    「動詞ない形」去掉「い」＋「くてもかまいません」 #

    原形 ない形 ない形許可句 解釋
    \(\overset{よ}{読}む\) \(\overset{よ}{読}\textcolor{blue}{まない}\) \(\overset{よ}{読}\textcolor{blue}{まなくてもかまいません}\) 不讀也沒關係
    \(\overset{い}{行}く\) \(\overset{い}{行}\textcolor{blue}{かない}\) \(\overset{い}{行}\textcolor{blue}{かなくてもかまいません}\) 不去也沒關係
    \(\overset{た}{食}べる\) \(\overset{た}{食}べない\) \(\overset{た}{食}べ\textcolor{blue}{なくてもかまいません}\) 不吃也沒關係
    \(\overset{おぼ}{覚}える\) \(\overset{おぼ}{覚}えない\) \(\overset{おぼ}{覚}え\textcolor{blue}{なくてもかまいません}\) 不記住也沒關係
    \(する\) \(\textcolor{blue}{しない}\) \(\textcolor{blue}{しなくてもかまいません}\) 不做也沒關係
    \(\overset{く}{来}る\) \(\textcolor{blue}{\overset{こ}{来}ない}\) \(\textcolor{blue}{\overset{こ}{来}なくてもかまいません}\) 不來也沒關係

    問句和回答 #

    只要在句尾加上表示疑問的「か」，就能夠直接當成疑問句使用

    問句：

    1. ～なければなりませんか。
    2. ～てもいいですか。
    3. ～なくてもいいですか。
    4. ～てもかまいませんか。
    5. ～なくてもかまいませんか。

    回答：

    • 肯定句的回答方式：重覆疑問句
    • 否定句的回答方式：轉一圈，使用其他句型

    句型比較 #

    同一個句子，三種不同問法

    1. 不寫作業不行嗎？ #

    \(\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなければなりませんか}。\)

    \(はい、\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなければなりません}。\)
    （是的，不寫作業不行）

    \(いいえ、\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなくてもいいです}。\)
    （不，不寫作業也可以）

    2. 不寫作業也可以嗎？ #

    \(\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなくてもいいですか}。\)

    \(はい、\textcolor{blue}{しなくてもいいです}。\)
    （是的，不寫作業也可以）

    \(いいえ、\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなければなりません}。\)
    （不，不寫作業不行）

    3. 不寫作業也沒關係嗎？ #

    \(\overset{しゅく}{宿}\overset{だい}{題}を\textcolor{blue}{しなくてもかまいませんか}。\)

    \(はい、\textcolor{blue}{しなくてもかまいません}。\)
    （是的，不寫作業也沒關係）
    \(いいえ、\overset{しゅく}{宿}\overset{だい}{題}を、\textcolor{blue}{しなければなりません。}。\)
    （不，不寫作業不行）

    其他例句 #

    可以看電視嗎？

    \(テレビを\textcolor{blue}{\overset{み}{見}てもいいですか}。\)

    \(はい、テレビをいいえ、テレビを\textcolor{blue}{\overset{み}{見}てもいいですよ}。 \)
    (是的，可以看電視喔)
    \(いいえ、テレビをいいえ、テレビを\textcolor{blue}{\overset{み}{見}てはいけません}。 \)
    (不，不行看電視)

    看電視也沒關係嗎？

    \(テレビを\overset{み}{見}\textcolor{blue}{てもかまいませんか}。\)

    \(はい、テレビを\textcolor{blue}{\overset{み}{見}てもかまいません}。 \)
    （是的，看電視也沒關係）
    \(いいえ、テレビを\textcolor{blue}{\overset{み}{見}てはいけません}。 \)
    （不，不行看電視）

    不吃青椒不行嗎？

    \(ピーマンを\textcolor{blue}{\overset{た}{食}べなければなりませんか}。\)

    \(はい、ピーマンを\textcolor{blue}{\overset{た}{食}べなければなりません}。 \)
    （是的，不吃青椒不行）
    \(いいえ、ピーマンを\textcolor{blue}{\overset{た}{食}べなくてもいいです}。 \)
    （不，不吃青椒也可以）
