    形容詞分類 #

    1. 以假名い結尾的形容詞
    2. 不以假名い結尾的形容詞
    3. 特例

    以假名い結尾的形容詞就叫い形容詞
    不以假名い結尾的形容詞則是な形容詞

    い形容詞 #

    以假名い結尾的形容詞 #

    日文 中文
    長(ながい) 長的
    短(みじかい) 短的
    広(ひろい) 寬的
    狭(せまい) 窄的
    深(ふかい) 深的
    浅(あさい) 淺的
    おいしい 好吃的

    句型 #

    い形容詞＋名詞 #

    • 長い傘: 很長的傘
    • 短い鉛筆: 很短的鉛筆
    • 広い部屋: 很寬的房間
    • 狭い部屋: 很窄的房間
    • 深い湖: 很深的湖
    • 浅い川: 很淺的河
    • おいしいデザート: 很好吃的甜點

    な形容詞 #

    不以假名い結尾的形容詞 #

    日文 中文
    静(しずか) 安靜的
    親切(しんせつ) 親切的
    賑やか(にぎやか) 熱鬧的
    便利(べんり) 方便的
    元気(げんき) 有精神的
    簡単(かんたん) 簡單的
    上手(じょうず) 拿手的
    下手(へた) 不拿手的

    句型 #

    な形容詞＋な＋名詞 #

    • 静かな教室: 安靜的教室
    • 親切な先生: 親切的老師
    • 賑やかなスーパー: 熱鬧的超市
    • 便利な所: 方便的地方
    • 元気な子ども: 朝氣十足的小孩
    • 簡単な質問: 簡單的問題
    • 上手なコック: 厲害的廚師
    • 下手な選手: 能力不佳的選手

    特例 #

    寫成漢字時，後面並不會出現「い」 #

    雖然寫成假名時是以「い」結尾 但是寫成漢字時，後面並不會出現「い」

    • 有名(ゆうめい): 有名的
    • 綺麗(きれい): 漂亮的

    有名な歌手: 有名的歌手 きれいな公園: 乾淨漂亮的公園

    「い」結尾，但仍屬於「な形容詞」 #

    • 嫌い(きらい): 討厭的

    嫌いな店員: 討厭的店員
