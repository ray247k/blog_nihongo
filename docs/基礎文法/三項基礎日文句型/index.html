    日文三種最基本的句型 #

    1. 肯定句
    2. 否定句
    3. 疑問句

    肯定句 #

    （名詞）は（名詞）です。 #

    \(\overset{わたし}{私}は\overset{せいがく}{学生}です。\) (我是學生)

    \(あなたは\overset{きょう}{教}\overset{し}{師}です。\) (你是老師)

    \(\overset{かれ}{彼}は\overset{しゃいん}{社員}です。\) (他是社員)

    \(\overset{かのじょ}{彼女}は\overset{しゃ}{社}\overset{ちょう}{長}です。\) (她是社長)

    否定句 #

    （名詞）は（名詞）ではありません。 #

    \(\overset{わたし}{私}は\overset{せいがく}{学生}ではありません。\) (我不是學生)

    \(あなたは\overset{きょう}{教}\overset{し}{師}ではありません。\) (你不是老師)

    \(\overset{かれ}{彼}は\overset{しゃいん}{社員}ではありません。\) (他不是社員)

    \(\overset{かのじょ}{彼女}は\overset{しゃ}{社}\overset{ちょう}{長}ではありません。\) (她不是社長)

    疑問句 #

    在肯定句「～は～です。」的最後加上表示問句的疑問詞「か」，就變成了疑問句
    「か」相當於中文中的「嗎、呢、吧」，用於問句的句尾

    發問 #

    （名詞）は（名詞）ですか。 #

    \(あなたは\overset{きょう}{教}\overset{し}{師}ですか。\) (你是老師嗎)

    \(\overset{かれ}{彼}は\overset{しゃいん}{社員}ですか。\) (他是社員嗎)

    \(\overset{かのじょ}{彼女}は\overset{しゃ}{社}\overset{ちょう}{長}ですか。\) (她是社長嗎)

    回答 #

    是的，沒錯 #

    はい、そうです。 #

    不，不是 #

    いいえ、違ちがいます。 #

    例句 #

    問題 #

    \(あなたは\overset{きょう}{教}\overset{し}{師}ですか。\) (你是老師嗎)

    是的 #

    はい、そうです。(是，沒錯)

    \(はい、\overset{わたし}{私}は\overset{きょう}{教}\overset{し}{師}です。\) (是，我是老師)

    不是 #

    \(いいえ、\overset{ちが}{違}います。\) (不，不是。)

    \(いいえ、 \overset{わたし}{私}は \overset{きょう}{教}\overset{し}{師}ではありません。\) (不，我不是老師)

    \(いいえ、 \overset{わたし}{私}は\overset{がくせい}{学生}です。\) (不，我是學生)
